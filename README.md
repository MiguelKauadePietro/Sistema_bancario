# Sistema Bancário em C++

Sistema bancário desenvolvido em **C++** com foco na aplicação de conceitos de **Programação Orientada a Objetos (POO)**, como classes, encapsulamento, herança e abstração.

O projeto simula operações básicas de um sistema bancário em ambiente de terminal, sendo voltado principalmente para fins educacionais e prática da linguagem.

## Objetivo do projeto
Este projeto foi criado com o objetivo de:
- praticar Programação Orientada a Objetos em C++
- entender a organização de um sistema baseado em classes
- aplicar conceitos de modelagem de entidades do mundo real (contas, clientes, operações)
- reforçar lógica de programação e estruturação de código

## Tecnologias utilizadas
- **C++**
- Compilador **g++**
- Ambiente Linux / terminal

## Funcionalidades implementadas
- Criação de contas bancárias
- Consulta de saldo
- Depósito
- Saque
- Estruturação do sistema utilizando classes e métodos

*(As funcionalidades podem variar conforme a versão do código.)*

## Conceitos de POO aplicados
- Classes e objetos
- Encapsulamento
- Métodos públicos e privados
- Organização em arquivos `.h` e `.cpp`

## Como compilar e executar

### Pré-requisitos
- g++ instalado

### Compilação
Certifique-se de que todos os arquivos `.cpp` e `.h` estejam no mesmo diretório.

```bash
g++ *.cpp *.h -o sistema_bancario
```

### Execução
```bash
./sistema_bancario
```
